# RainMark - 智能书签管理插件

RainMark 是一个功能强大的 Chrome 浏览器扩展，提供智能化的书签管理功能。它通过人工智能技术自动分类书签、推荐相关内容，并提供多种优化工具，让您的书签管理更加高效。

## ✨ 主要功能

### 🏷️ 智能分类
- **自动分类**: 基于书签标题和URL内容自动分类到相应文件夹
- **自定义规则**: 支持用户自定义正则表达式分类规则
- **多语言支持**: 支持中文和英文关键词识别
- **默认分类**: 包含工作、学习、娱乐、新闻、技术、购物等常见分类

### 🔍 智能搜索与推荐
- **模糊搜索**: 支持标题和URL的模糊匹配搜索
- **语义推荐**: 根据当前浏览页面推荐相关书签
- **历史分析**: 基于浏览历史计算书签相关性
- **实时推荐**: 在浏览相关页面时显示推荐通知

### 🧹 书签清理与优化
- **重复检测**: 自动检测并清理重复书签
- **失效链接检查**: 定期检查书签链接是否有效
- **自动归档**: 将失效链接移动到专用文件夹
- **使用统计**: 显示书签分类统计和使用情况

### 🎨 美观的用户界面
- **现代化设计**: 采用现代化UI设计，支持深色/浅色主题
- **响应式布局**: 适配不同屏幕尺寸
- **直观操作**: 拖拽排序、批量操作等便捷功能
- **实时反馈**: 操作成功/失败的通知提示

## 📦 安装方法

### 从源码安装
1. 克隆或下载本项目到本地
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目目录中的 `RainMark` 文件夹

## 🚀 快速开始

### 基本使用
1. **添加书签**: 正常添加书签时，RainMark会自动进行分类
2. **查看分类**: 书签栏中会自动创建分类文件夹
3. **搜索书签**: 点击扩展图标，在搜索框中输入关键词
4. **查看推荐**: 浏览网页时，系统会自动推荐相关书签

### 高级功能
1. **自定义分类规则**: 在设置页面添加自定义正则表达式规则
2. **导出/导入书签**: 支持JSON格式的书签备份和恢复
3. **定时清理**: 设置自动清理重复书签和检查失效链接
4. **统计分析**: 查看书签使用情况和分类统计

## 🛠️ 技术架构

### 核心技术
- **Chrome Extension API**: 使用 Chrome 扩展API进行书签操作
- **JavaScript ES6+**: 现代JavaScript语法
- **HTML5/CSS3**: 响应式网页设计
- **正则表达式**: 用于文本匹配和分类

### 智能算法
- **关键词匹配**: 基于关键词权重的分类算法
- **相似度计算**: Jaccard相似度算法
- **历史分析**: 基于浏览历史的推荐算法
- **模式识别**: 正则表达式模式匹配

### 数据存储
- **chrome.storage**: 存储用户设置和规则
- **本地缓存**: 临时数据存储
- **JSON导出**: 数据备份格式
