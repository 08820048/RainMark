# RainMark 智能书签管理插件 - 安装指南

## 📋 系统要求

- **操作系统**: Windows 10/11, macOS 10.14+, Linux (支持 Chrome 的系统)
- **浏览器**: Google Chrome 88 或更高版本
- **磁盘空间**: 至少 10MB 可用空间
- **内存**: 建议 2GB 以上可用内存

## 📥 安装方法

### 方法一：从 Chrome 网上应用店安装（推荐）

1. 打开 Chrome 浏览器
2. 访问 [Chrome 网上应用店](https://chrome.google.com/webstore)
3. 搜索 "RainMark" 或 "智能书签管理"
4. 点击 "添加到 Chrome" 按钮
5. 在弹出的确认对话框中点击 "添加扩展程序"

### 方法二：从源码安装（开发者/测试版）

#### 步骤 1：下载源码

**选项 A：从 GitHub 下载**
```bash
# 使用 git 克隆仓库
git clone https://github.com/yourusername/RainMark.git

# 或者直接下载 ZIP 文件
# 1. 访问 GitHub 仓库页面
# 2. 点击 "Code" → "Download ZIP"
# 3. 解压下载的文件
```

**选项 B：从本地项目安装**
如果您已经拥有 RainMark 项目文件，请确保目录结构如下：
```
RainMark/
├── manifest.json
├── background.js
├── popup.html
├── popup.js
├── options.html
├── options.js
├── styles.css
├── icons/
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
└── README.md
```

#### 步骤 2：加载扩展程序

1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 开启右上角的 **"开发者模式"** 开关
4. 点击左上角的 **"加载已解压的扩展程序"** 按钮
5. 在弹出的文件选择对话框中，导航到 RainMark 项目文件夹
6. 选择包含 `manifest.json` 文件的目录
7. 点击 **"选择文件夹"**

#### 步骤 3：验证安装

1. 在扩展程序页面中，您应该能看到 RainMark 扩展
2. 确保扩展旁边的开关处于 **"启用"** 状态
3. 您可以在 Chrome 工具栏中看到 RainMark 的图标

## 🔧 初始设置

### 首次运行配置

1. **点击扩展图标**：在 Chrome 工具栏中点击 RainMark 图标
2. **授予权限**：根据提示授予必要的权限：
   - ✅ 书签权限（必需）
   - ✅ 存储权限（必需）
   - ✅ 标签页权限（推荐）
   - ✅ 历史记录权限（可选，用于智能推荐）
   - ✅ 通知权限（可选，用于接收通知）

3. **基本设置**：
   - 打开扩展的选项页面（右键点击扩展图标 → "选项"）
   - 配置自动分类、通知等设置
   - 点击 "保存设置"

### 权限说明

| 权限 | 必要性 | 用途 |
|------|--------|------|
| `bookmarks` | 必需 | 读取、创建、修改书签 |
| `storage` | 必需 | 保存用户设置和分类规则 |
| `tabs` | 推荐 | 获取当前标签页信息，用于智能推荐 |
| `history` | 可选 | 分析浏览历史，优化推荐算法 |
| `notifications` | 可选 | 显示操作成功/失败通知 |
| `alarms` | 自动 | 定时执行任务（如检查失效链接） |

## 🚀 快速开始

### 1. 添加第一个书签

1. 正常添加书签（Ctrl+D 或点击地址栏的星标）
2. RainMark 会自动分析书签内容
3. 书签会被分类到相应的文件夹中
4. 您会收到分类成功的通知

### 2. 使用智能搜索

1. 点击 RainMark 扩展图标
2. 在搜索框中输入关键词
3. 系统会显示相关的书签
4. 点击书签即可在新标签页中打开

### 3. 查看智能推荐

1. 浏览网页时，RainMark 会自动分析页面内容
2. 如果发现相关书签，会显示推荐通知
3. 点击通知可以查看推荐的书签列表

### 4. 管理书签分类

1. 在选项页面可以：
   - 查看现有的分类规则
   - 添加自定义分类规则
   - 调整分类设置
   - 导出/导入分类规则

## 🛠️ 故障排除

### 常见问题

#### 问题 1：扩展图标不显示
- **解决方案**：
  1. 点击 Chrome 工具栏的扩展程序图标（拼图形状）
  2. 找到 RainMark 并点击图钉图标固定到工具栏
  3. 如果仍然不显示，重启 Chrome 浏览器

#### 问题 2：权限被拒绝
- **解决方案**：
  1. 前往 `chrome://extensions/`
  2. 找到 RainMark 扩展
  3. 点击 "详细信息"
  4. 确保所有必要权限都已启用
  5. 如果权限被拒绝，可能需要重新安装扩展

#### 问题 3：分类功能不工作
- **解决方案**：
  1. 检查是否启用了自动分类功能
  2. 打开选项页面，确保 "启用自动分类" 已勾选
  3. 检查分类规则是否正确
  4. 尝试重新启动扩展

#### 问题 4：推荐功能不显示
- **解决方案**：
  1. 确保已启用智能推荐功能
  2. 检查是否授予了标签页和历史记录权限
  3. 等待一段时间让系统收集足够的浏览数据

### 调试模式

如需更详细的错误信息，可以启用调试模式：

1. 打开 Chrome 开发者工具（F12）
2. 切换到 "Console" 标签页
3. 查看是否有 RainMark 相关的错误信息
4. 可以在选项页面开启 "调试日志"（如果支持）

## 🔄 更新与维护

### 手动更新（源码安装）

1. 下载最新版本的源码
2. 前往 `chrome://extensions/`
3. 找到 RainMark 扩展
4. 点击 "刷新" 按钮（或先删除再重新加载）

### 自动更新（Chrome 网上应用店）

从 Chrome 网上应用店安装的扩展会自动更新。

### 数据备份

建议定期备份您的书签和设置：

1. **导出书签**：
   - 在选项页面点击 "导出书签"
   - 保存生成的 JSON 文件到安全位置

2. **导出设置**：
   - 分类规则会自动保存在 Chrome 同步存储中
   - 如需手动备份，可以导出分类规则
